# 📊 Modelo de Ising em 2D

## 📘 Descrição

Algoritmo de Metropolis implementado em Python aplicado na resolução do Modelo de Ising em 2D, numa malha 20x20 com a configuração de 25% de spins up (↑) e 75% de spins down (↓).
